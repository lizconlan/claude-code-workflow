# Sprint 6: Accessibility Pass + Polish

## Goal
Ensure the app meets WCAG AA and looks polished.

## Tasks

- [ ] Verify all `aria-label` attributes on cells update correctly (e.g. "Cell 0, X")
- [ ] Confirm `aria-live="polite"` on #status announces moves to screen readers
- [ ] Check all colour contrast ratios are ≥ 4.5:1
- [ ] Confirm focus-visible outline is visible on keyboard navigation
- [ ] Confirm hover underline on any text links (none expected, but verify)
- [ ] Verify font sizes are readable at default zoom (≥ 16px body, ≥ 2rem cells)
- [ ] Disable pointer cursor on occupied/game-over cells
- [ ] Final visual review: consistent fonts, colours, spacing
