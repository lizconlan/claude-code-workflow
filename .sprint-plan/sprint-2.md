# Sprint 2: CSS Styling

## Goal
Style the application using the design tokens defined in SPEC.md.

## Tasks

- [ ] Define CSS custom properties (design tokens) on `:root`
- [ ] Style `body`: background, font, centred layout
- [ ] Style `h1` header
- [ ] Style `#scoreboard`: flex row, evenly spaced score items
- [ ] Style `#status`: clear, readable turn/result message
- [ ] Style `#board`: 3×3 CSS Grid
- [ ] Style `.cell` buttons: size, border, font, cursor, hover, focus-visible
- [ ] Style `.win` class: winning cell highlight
- [ ] Style `#reset-btn`: filled primary colour, hover state
- [ ] Ensure colour contrast meets WCAG AA (4.5:1)
